geobabel
========

Fast conversion between GIS formats. Currently in alpha stage and supports:

Formats:

- WKB (read / write)
- WKT (write)
- GML (with [`geobabel-gml`](https://github.com/charto/geobabel-gml))

Geometries:

- Point
- LineString
- Polygon
- MultiPoint
- MultiLineString
- MultiPolygon
- GeometryCollection
- MultiCurve
- MultiSurface
